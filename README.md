# PARKPULSE AI
AI powered parking intelligence for hotspot detection and Smart enforcement

# PROBLEM STATEMENT
Detect illegal parking hotspots and quantify their impact on traffic congestion using AI-driven parking intelligence.

# FEATURES
DBSCAN-based hotspot detection
Congestion Impact Score (CIS)
Random Forest prediction model
Smart enforcement recommendations
Interactive dashboard using Flask

# TECH-STACK 
Python
Pandas
Scikit-Learn
DBSCAN
Folium
Flask
Bootstrap
